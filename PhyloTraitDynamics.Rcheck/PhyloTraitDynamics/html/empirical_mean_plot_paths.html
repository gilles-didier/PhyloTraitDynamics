empirical_mean_plot_paths {PhyloTraitDynamics}R Documentation

Plot Empirical Mean Paths

Description

Plots individual simulated empirical mean paths.

Usage

empirical_mean_plot_paths(
  simulation,
  max_paths = 100,
  alpha = 0.08,
  main = NULL,
  xlab = "time",
  ylab = "Empirical mean"
)

Arguments

simulation

An "empirical_mean_simulation" object.

max_paths

Maximum number of simulated paths to draw.

alpha

Transparency used for simulated paths.

main, xlab, ylab

Base graphics labels.

Value

Invisibly returns simulation.
