empirical_variance_summarise_simulation {PhyloTraitDynamics}R Documentation

Summarise an Empirical Variance Simulation

Description

Computes Monte Carlo summaries from a simulation returned by birth_death_brownian_simulate().

Usage

empirical_variance_summarise_simulation(x)

Arguments

x

An "empirical_variance_simulation" object.

Value

A data frame with time, empirical survival probability, unconditional Monte Carlo mean empirical variance, and survival-conditioned Monte Carlo mean empirical variance.
