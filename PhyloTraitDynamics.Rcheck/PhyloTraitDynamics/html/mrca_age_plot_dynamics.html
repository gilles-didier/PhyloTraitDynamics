mrca_age_plot_dynamics {PhyloTraitDynamics}R Documentation

Plot MRCA Age Distribution Dynamics

Description

Plots MRCA age summaries over time, including median, optional interquartile band, and optional expectation.

Usage

mrca_age_plot_dynamics(
  x,
  show_iqr = TRUE,
  show_expectation = TRUE,
  main = NA,
  xlab = "Time",
  ylab = "MRCA age",
  add_legend = TRUE,
  ...
)

Arguments

x

An "mrca_age_dynamic" object.

show_iqr

Logical; whether to draw the interquartile band.

show_expectation

Logical; whether to draw the expectation curve.

main, xlab, ylab

Base graphics labels.

add_legend

Logical; whether to draw the plot legend.

...

Additional arguments passed to graphics::plot().

Value

Invisibly returns x.
